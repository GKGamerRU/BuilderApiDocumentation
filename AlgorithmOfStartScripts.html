Builder API

Алгоритм инициализации скриптов.

При инициализации игры

Сначала инициализируются скрипты из папки мода:
Происходит загрузка скрипта script.cs
Происходит загрузка скрипта OnMapRedraw.cs
Происходит загрузка скрипта OnGlobalTimerTick.cs
Происходит загрузка скрипта OnAbout.cs

Затем инициализируются скрипты из папки блока:
Происходит загрузка скрипта script.cs
Происходит загрузка скрипта OnSelect.cs
Происходит загрузка скрипта OnRedraw.cs
Происходит загрузка скрипта OnDestroy.cs
Происходит загрузка скрипта OnClick.cs

Происходит инициализация script.cs, после загрузки всех предыдущих скриптов.
Начинается выполнение OnGlobalTimerTick.cs

При Перерисовке

Приостанавливается выполнение OnGlobalTimerTick.cs
Происходит выполнение скрипта OnRedraw.cs
Происходит выполнение скрипта OnMapRedraw.cs после завершения перерисовки
Возобновляется выполнение OnGlobalTimerTick.cs

При загрузке карты

Приостанавливается выполнение OnGlobalTimerTick.cs
Происходит выполнение скрипта OnRedraw.cs
Происходит выполнение скрипта OnMapRedraw.cs после завершения перерисовки
Возобновляется выполнение OnGlobalTimerTick.cs
