Builder API

ModManager

Mods - Список всех инициализированных модов в билдере. (List of Mod)
GetMod(string name) - Возвращает мод с указанным именем. (Mod)

Mod - Класс

Name - Имя мода. (string)
Path - Путь к папке с модом. (string)
ResourcePath - Путь к папке с ресурсами в моде. (string)
Blocks - Список блоков мода. (List of ModG)
GetBlock(string name) - Возвращает блок мода по имени. (ModG)
Данный класс способен работать с Properties и Resources: Properties&Resource
