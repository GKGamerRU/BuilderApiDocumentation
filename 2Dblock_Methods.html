Builder API

_2Dblock Methods

GetBlocks() - Получает все поставленные блоки. (List of Block)
GetBlockByIDTexture(int id) - можно получить массив блоков по указанному id текстуры. (List of Block)
GetBlockByID(int id) - можно получить блок с указанным id из массива блоков. (Block)
GetBlockByName(string name) - можно получить первый по очереди блок по указанному имени из массива блоков. (Block)
GetBlockByNameArray(string name) - можно получить массив блоков по указанному имени. (IEnumerable of Block)
GetBlockByModNameArray(string name) - получение объекта по имени мода, из которого блок. (IEnumerable of Block)
GetGFX() - получает Graphics. (Graphics)
PaintTable() - Получает панель, в которой на PictureBox рисуются блоки. (Panel)
LoadPanel() - получает панель загрузки карты. (Panel)
Form() - Получает форму, в которой запущен 2Д билдер. (Form)
GetBackground() - Получает текущий BackgroundImage. (Image)
GetCounter() - Получает зеленый PictureBox. При поставке блоков игра использует его координаты. (Control as PictureBox)
GetSelectedInventar() - получает выбранный в данный момент инвентарь(слот). (Inventar)
GetThisBlock() - получает блок, в котором выполнен скрипт. (Block)
GetCurrentModInfo() - Получает информацию о моде в выбранном в инвентаре блоке. (string)
Destroy(int x, int y) - Уничтожает блок по заданной координате.
Destroy(Block obj) - Уничтожает выбранный блок.
DestroyAll() - Очищает карту от блоков.
TryBuild(int x, int y, ModG pattern) - Ставит блок, если координата не занята. (Block)
Build(int x, int y, ModG pattern) - Ставит Блок из паттерна в заданную координату даже если там уже стоит блок. (Block)
Render() - Рендерит, то есть применяет перерисовку всех объектов.
AddTool(Bitmap icon, Action onSelect, Action onDeselect) - Добавляет новый инструмент в список и возвращает его.
RemoveTool(ToolButton tool) - Убирает инструмент из списка.[Не рекомендуется]
