Builder API

_2Dblock Classes

Для того, чтобы понимать, что за классы в апи, вот список и их поля:

Block

OnRedraw - Происходит при перерисовке блока. (Action)
OnDestroy - Происходит при уничтожении блока. (Action)
OnClick - Происходит при нажатии на блок. (Action)

Position - Позиция блока. (Vector2)
Scale - Размер блока. (Vector2)
Texture - текстура блока. (Image)
(Устарело) _IndexTexture - Индекс текстуры встроенного блока. (int)
Mod - Мод, которому блок принадлежит. (ModManager.Mod)
BlockMod - Паттерн блока. (ModG)
IsInteractive - Возвращает true, если OnClick не равен null. (bool)

Properties - CustomProperties (Dictionary|string, string|)

Execute(string code) - Выполняет скрипт из текста (Method)
draw() - Нарисовать блок на сцене (Method)

Данный класс способен работать с Properties: Properties&Resource

ModG

OnBlockPlaced - срабатывает при появлении блока (Action)
OnDestroyed - Срабатывает перед уничтожением блока (Action)
OnBlockSelect - Срабатывает при выборе блока в инвентаре (Action)
OnRedraw - Происходит при перерисовке блока (Action)
OnClick - Срабатывает при клике на блок (Action)

Mod - мод, которому блок принадлежит (ModManager.Mod)
ModName - Имя мода (string)
_Texture - Текстура блока (Image)
BlockName - Имя блока (string)
