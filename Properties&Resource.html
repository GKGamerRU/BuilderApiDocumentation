Builder API

Custom & Global properties

Это подходит для блока и мода. В блоке при сохранении карты свойства сохраняются, в отличии от мода.
Свойства в Global хранятся в виде object, а в Custom в виде string листа.
Custom Property - Block
Global Property - Mod

Для мода (Global)

SetProperty(string key, object content) - ставит свойство.
GetProperty(string key) - возвращает свойство object.

Для Блока (Custom)

SetProperty(string key, string content) - ставит свойство.
GetProperty(string key) - возвращает свойство string.

Если вы пытаетесь получить свойство, которого нет, то возвращается null!

Resource System

В каждом моде есть ресурсы благодаря новой системе. Для нее надо создать в папке мода папку "Resources"
Также блока "Resources" не может быть, так как это название зарезервировано для этой системы

Методы

GetTextFromResources(string FileName) - Возвращает string из файла
GetImageFromResources(string FileName) - Возвращает Image из файла
GetBytesFromResources(string FileName) - Возвращает byte[] из файла
GetSoundFromResources(string FileName) - Возвращает SoundPlayer со звуком из WAV файла
SetTextToResources(string FileName, string content) - Сохраняет string данные в файл
SetImageToResources(string FileName, Image content) - Сохраняет Image в файл
SetBytesToResources(string FileName, byte[] content) - Сохраняет byte[] в файл

Если вы пытаетесь получить свойство, которого нет, то возвращается null!
